3. ### Java

   类似于网页开发的JavaScript

   - 变量

     ```java
     int x = 10086;  //数值
     String name = "KingSwim" //文本
     boolean isEasy = true;
     ```

   - 分支

     ```java
     boolean isRich = true;
     if(isRich){
         //吃小龙虾
     }else{
         //吃土
     }
     ```

   - 循环
   
     ```java
     int x = 7;
     int factorial = 1;
     for(int i= 1 ; i <= x ; i = i+1){
    
       factorial = factorial * i;
    
      }
     ```

   - 方法

     ```java
     public int getFactorial(x){
      int factorial = 1;
      
      for(int i= 1 ; i<= x ; i = i+1){
      
       factorial = factorial * i;
      
        }
        return factorial;
      }
     ```
